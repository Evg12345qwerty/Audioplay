Как использовать:

  • Укажите репозиторий в формате username/repository
  • Введите путь к аудиофайлу в репозитории
  • Для публичных репозиториев файл должен быть в ветке main или master
  • GitHub Audio Player автоматически сгенерирует прямую ссылку на файл
  • Нажмите "Загрузить песню" для добавления в плейлист
